# Archived planning documents

The documents in this directory are point-in-time planning artifacts — assessments,
master plans, execution plans, and parity audits written while the ABCAC platform was
being built. They are kept for historical context and traceability of decisions, but
they were authored against much older code baselines and no longer describe the shipped
system. Do not use them as operational references; for the current state of the platform
see [`docs/LAUNCH-ASSESSMENT.md`](../LAUNCH-ASSESSMENT.md), and for day-to-day operations
use `SETUP-RUNBOOK.md` and `AUTOMATION-RUNBOOK.md` at the repo root.
